## 🏠 Open House Feedback Multi-Lable Classification

### Background
一筆物件的成交少不了多次的帶看，買方在看房後，不僅近一步了解房屋的實際狀況，也具象化自身夢寐以求房屋的想像。
本專案根據仲介輸入的買方帶看回饋，建立BERT多標籤分類模型，協助收斂出買方的彈性需求，進而影響固定剛性需求下的房屋配對排序，提升物件推薦的精準度。
